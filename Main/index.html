CSS Snippet Cheat Sheet

Inline Elements

Inline-level elements fill only the inline space (i.e. width) of the contents.

          
        a           button     img        
        
        input       label      span
        
        strong      textarea

Block Elements

Block-level elements fill the available inline space of the parent element.


        div         footer      form
        
        h1-h6       header      li
        
        main        nav         p                
        
        section     ul
